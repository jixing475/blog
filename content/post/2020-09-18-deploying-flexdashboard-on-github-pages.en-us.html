Github最酷的功能之一就是Github Pages，可以免费在github.com下创建和部署网站。Github Pages最常见的用法是部署项目文档。 R社区广泛使用它来部署不同的R Markdown格式:

  1. pkgdown的软件包文档
  2. blogdown的博客
  3. bookdown的图书等。

那么在Github Pages上部署flexdashboard应该也是可行的, 因为 flexdashboard 也是Rmd 的一种形式.

怎么做呢?

1. git 和 GitHub 账户(Done!)

2. flexdashboard 的项目文件https://github.com/jixing475/flexdashboard_example

  • 创建 flexdashboard 的 index.Rmd 文件
  • 创建_site.yml文件: ‘name: "flexdashboard-example", output_dir: docs’ 关于 _site.yml详细情况: 10.5 rmarkdown’s site generator | R Markdown: The Definitive Guide

3. 同步到 GitHub 并设置 Github page

注意生成的网址是: https://Your_Github_User_Name.github.io/Repository_Name/
